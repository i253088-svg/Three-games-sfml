# Gaming Hub 🎮

A desktop gaming application developed in **C++** using the **SFML (Simple and Fast Multimedia Library)** framework. The Gaming Hub provides a centralized platform where users can play multiple games, track their performance, and view detailed statistics.

## Features

* **Three Integrated Games**

  * Snake
  * Ping Pong
  * Tic-Tac-Toe

* **Object-Oriented Design**

  * Implemented using core OOP principles:

    * Encapsulation
    * Inheritance
    * Polymorphism
    * Abstraction
  * Modular and maintainable code structure.

* **Player Statistics Dashboard**

  * Tracks game performance and outcomes.
  * Displays statistics for each game.
  * Provides an overview of player progress.

* **File Handling**

  * Stores player data and game statistics.
  * Retrieves saved records when the application is launched.
  * Ensures persistence of user progress.

* **Graphical User Interface**

  * Built using SFML.
  * Interactive menus and navigation system.
  * Smooth user experience across all games.

## Technologies Used

* C++
* SFML (Simple and Fast Multimedia Library)
* Object-Oriented Programming (OOP)
* File Handling
